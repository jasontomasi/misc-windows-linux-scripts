SQL Server: reset password on SA account
-----------------------------------------

Run Command prompt with Administrator privileges

net stop MSSQL$GOLDMINE
(this stops the SQL server service)

net start MSSQL$GOLDMINE /f
(this starts the SQL server service in single-user mode)

sqlcmd -E -S HOSTNAME\GOLDMINE
(this connects to the instance so we can issue commands to it)

sqlcmd -E
(if you don't know the instance name, as per the previous step, then this command should allow you to connect to the default instance)

ALTER LOGIN sa enable
ALTER LOGIN sa WITH PASSWORD = ''
GO

(these commands set a new password for the 'sa' account)

CREATE LOGIN intuitive WITH PASSWORD = 'putYourNewPasswordHere';
ALTER SERVER ROLE sysadmin ADD MEMBER intuitive
EXEC sp_addsrvrolemember ‘intuitive, ‘sysadmin’;
GO

(you can also create a new SQL user account here)


RESTART THE SQL SERVER OR REBOOT THE MACHINE

net stop MSSQL$GOLDMINE
(this stops the SQL server service)

net start MSSQL$GOLDMINE
(this starts the SQL server service)
