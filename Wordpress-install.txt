Manual steps:
-------------

Add line to file "wp-config.php" to enable WordPress to download themes from Wordpress.org:
define( 'FS_METHOD','direct' ); 





chown -R www-data:www-data /var/www/
chmod -R 755 /var/www/

find /var/www/ -type d -exec chmod 755 {} \;
find /var/www/ -type f -exec chmod 644 {} \;

(Source: https://wordpress.org/support/article/hardening-wordpress/#changing-file-permissions)
